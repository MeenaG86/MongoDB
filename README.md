Overview

This project demonstrates the basic operations of MongoDB, including creating a database, inserting data, reading data, updating records, and deleting documents. It also includes the use of query operators and a simple real-world use case.

🚀 Technologies Used
MongoDB
MongoDB Shell (mongosh)
